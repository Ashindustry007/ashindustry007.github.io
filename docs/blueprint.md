# **App Name**: Panda Portfolio

## Core Features:

- Configurable Content Management: Allow effortless customization of all text, images, and links across the hero section and all portfolio content via a configuration.
- Scroll-Driven Parallax Hero: Implement a dynamic, full-screen WebP image sequence background that advances smoothly with scroll down and reverses with scroll up.
- Structured Portfolio Sections: Display dedicated sections for About, Experience & Research, Featured Projects, and Beyond AI, with layouts specified in the prompt.
- Dark Theme & Orange Accent: Apply a consistent dark mode aesthetic with the specified vibrant orange accent to maintain a premium and cinematic visual experience.
- Pre-Content Loading Experience: Provide a full-screen loading overlay with a dynamic progress bar, revealing the hero content only after all WebP frames are fully loaded.
- External Sub-Page Integration: Enable seamless navigation from main sections to external HTML pages for Photography, Extracurriculars, and Coursework.
- Dynamic Social & Navigation: Implement customizable social media icons and horizontal navigation links at the hero's bottom for easy access to related content.

## Style Guidelines:

- Primary accent color: Vibrant Orange (#FF8000), inspired by the user's request for a 'neon accent' and 'Black and Orange' theme, to highlight interactive elements and key information.
- Background color: Deep Charcoal (#1C1917), derived from the primary hue but heavily desaturated to create a sophisticated, dark 'cinematic aesthetic' foundation.
- Headline font: 'Space Grotesk' (sans-serif), chosen for its techy and scientific feel, aligning with expertise in AI, ML, and Software Engineering. Body text font: 'Inter' (sans-serif), providing excellent legibility and a modern, neutral look, paired for balance.
- Utilize minimal, monochrome social icons (GitHub, LinkedIn, Instagram) in line with the dark cinematic aesthetic, ensuring clarity and unobtrusive design.
- Employ a premium single-page parallax layout, respecting exact text and image composition for the hero section and adopting grid/card structures for other content sections to ensure a bold, modern feel.
- Ensure smooth, cinematic transitions for the scroll-driven parallax background, gentle fades for content updates, and fluid loading animations for a polished user experience.
